A Horta Comunitária foi desenvolvida como Trabalho de Conclusão de Curso (TCC), com o objetivo de integrar tecnologia e impacto social por meio de uma solução digital voltada à promoção da sustentabilidade e do engajamento comunitário.

O projeto propõe a criação de uma plataforma que facilita a organização, o gerenciamento e a divulgação de hortas comunitárias, incentivando práticas sustentáveis, alimentação saudável e a colaboração entre membros da comunidade.

🎯 Objetivo

O principal objetivo deste projeto foi desenvolver uma aplicação funcional que:

Incentive a participação comunitária

Promova a educação ambiental

Auxilie na gestão de hortas urbanas

Utilize a tecnologia como ferramenta de transformação social

💻 Desenvolvimento

Durante o desenvolvimento, foram aplicados conceitos fundamentais de desenvolvimento web, incluindo:

Estruturação de páginas com HTML

Estilização e responsividade com CSS

Implementação de funcionalidades interativas com JavaScript

O projeto foi pensado para simular um sistema real, focando na usabilidade, organização das informações e experiência do usuário.

🌍 Impacto

A Horta Comunitária busca contribuir para:

O fortalecimento de vínculos sociais

A conscientização sobre sustentabilidade

O acesso a alimentos mais saudáveis

🛠️ Tecnologias Utilizadas

HTML5

CSS3

JavaScript
